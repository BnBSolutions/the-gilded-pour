# The Gilded Pour

## GitHub Pages deploy

Repo-ul este pregatit pentru deploy automat pe GitHub Pages.

1. Fa push pe branch-ul `main`.
2. In GitHub mergi la `Settings > Pages`.
3. La `Source`, alege `GitHub Actions`.
4. Dupa primul workflow reusit, site-ul va fi disponibil la `https://bnbsolutions.github.io/the-gilded-pour/`.
